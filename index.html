Úvod do CSS

Cascading Style Sheets

Kaskádové štýly - jedná sa o programovací jazyk pre vizuálnu štylizáciu výstupu kódu HTML. Neustále sa vyvýja a pribúdajú nové možnosti štýlovania.

Niektore vlastnosti sú dedičné a potomkovia ich automaticky dedia od rodičov - color, font, ...

Pozor

Nie všetky prehliadače podporujú všetky vlastnosti. Pre použitie niektorých vlastností je potrebné vložiť pred názov vlastnosti predponu. Existujú spôsoby automatického vytvárania štýlu s predponami.
  • -webkit - prehliadač Chrome a Safari
  • -moz - prehliadač Firefox

Podporu na prehliadačoch s overíš na Can I use

Ukážka

  • -webkit-box-shadow: 10px 10px 5px 0px rgba(0,0,0,0.75);
  • -moz-box-shadow: 10px 10px 5px 0px rgba(0,0,0,0.75);
  • box-shadow: 10px 10px 5px 0px rgba(0,0,0,0.75);

Vloženie štýlov do kódu

Externé

CSS sa píše v samostatnom súbore, ktorý sa následne pomocou tagu vkladá do HTML

<link href="/style.css" rel="stylesheet">

Interné

CSS sa píše priamo v HTML vo vnútry tagu

<style> body { background-color: lightblue; } </style>

Inline

CSS sa píše priamo v príslušnom tagu

<h1 style="font-size: 40px; text-transform: uppercase"> </h1>

Resetovanie výchozích štýlov

Niktoré tagy majú v niektorých prehliadačoch preddefinované vlastnosti. Tieto vlastnosti je vhodné na začiatku zresetovať

Meyer CSS reset

CSS Selektori

Aby bolo možné určitý tag naštýlovať je potrebné ho správne vyselektovať. Je možné vyselektovať samotný tag, jeho potomkov alebo jeho súrodenca. Nie je možné selektovať rodičov!!! Kaskádové štýly idú len cestou nadol, nikdy nie opačne.

Zoznam selektorov: Selektori

Vybrané selektory:

Tag

Selektovanie pomocou tagu využívame len v špeciálnych prípadoch. Tagy sa opakujú naprieč celý webom a priradení vlastnosti tagu <div> sa naštýlujú všetky <div> tagy.

Použitie: <h1> - <h6> - používanie jednotného štýl pre nadpisy rovnakej úrovne napomáha jednoduchšej orientácii užívateľa na webe

h1 {
font-size: 40px;
text-transform: uppercase;
}

Class

Atribut class je najbežnejším a najvhodnejším selektorom, jeden tag môže mať viacero class a jednu class môže mať viac tagov

.class {
font-size: 40px;
text-transform: uppercase;
}

Id

Atribut id je unikátnym selektorom, jeden tag môže mať len jeden id atribut a žiaden ďalší tag nesmie mať rovnaký id atribut. Ako CSS selektor sa pre svoju unikátnosť využíva minimálne. Využíva sa ako kotva v odkazoch alebo pre selektovanie v JavaScripte.

#id {
font-size: 40px;
text-transform: uppercase;
}

:first-child / :last-child

Pomocou týchto selektorov je možné vyselektovať prvého alebo posledného potomka daného rodiča.

HTML
<ul>
<li> Prvý potomok </li>
<li> Druhý potomok </li>
<li> Tretí potomok </li>
<li> Posledný potomok </li>
</ul>
CSS
li:first-child {
color: #fff;
}

:nth-child()

Umožňuje vyselektovať akéhokoľvek potomka podľa jeho poradia. Bežne sa nepoužíva - potomkovia môžu pribúdať/ubúdať a poradové číslo potom neodpovedá.

HTML
<ul>
<li> Prvý potomok </li>
<li> Druhý potomok </li>
<li> Tretí potomok </li>
<li> Posledný potomok </li>
</ul>
CSS
li:nth-child(3) {
color: #fff;
}

:nth-child(even), :nth-child(even),

Umožňuje vyselektovať všetkých nepárnych / párnych potomkov.

HTML
<ul>
<li> Prvý potomok </li>
<li> Druhý potomok </li>
<li> Tretí potomok </li>
<li> Posledný potomok </li>
</ul>
CSS
li:nth-child(even) {
color: #fff;
}

Selektory je možné vzájomne kombinovať

Selektory oddelené medzerou

Selektory oddelené medzerou odpovedajú zanoreniu jednotlivých tagov. Na začiatku rady je Rodič a na konci posledný potomok

HTML
<div id="container">
<div class="section">
<ul class="list">
<li> Prvý potomok </li>
<li> Druhý potomok </li>
<li> Tretí potomok </li>
<li> Posledný potomok </li>
</ul>
</div>
</div>
CSS
#container .section .list li:last-child {
color: #fff;
}

Selektory oddelené čiarkou

Selektory oddelené čiarkou sú samostatnými selektormi, ktoré majú rovnaký štýl.

HTML
<div class="section">
<p class="paragraph"> Prvý vyselektovaný prvok</p>
</div>

<div class="container">
<h2> Druhý vyselektovaný prvok</p>
</div>
CSS
.section .paragraph, container h2 {
color: #fff;
}

Sila selektorov

Jednotlivé selektory majú rozdielnu silu a vzájomne sa prebíjajú. V prípade kombinácie selektrov sa ich sila spočítava. V prípade selektorov s rovnakou silou, prevládne vlastnosť napísané neskôr v kóde.

  • Tag - najnižšia sila
  • Class - silnejší
  • ID - silnejší
  • Inline štýl - prebije všetky ostatné selektory
  • !important - nasilnejší, prepíše aj inline štýl. Je ho možné prepísať len ďalším 1important. Využíva sa len k krajných prípadoch (napr. prepísanie štýlu šablóny, utility)
div {color: blue;}
.section {color: blue;}
#test {color: blue;}
<div style="color: blue;"> Inline štýl</div>
div {color: blue!important;}

Jednotky

Zoznám základných jednotiek, ktoré sa bežne používajú:

  • px
  • rem - jednotka relatívna k veľkosti písma na html tagu, zvyčajne 16px
  • em - jednotka relatívna k veľkosti písma na selektovanom tagu, 2em = 2x veľkosť písma
  • %
  • vh - jednotka relatívna k šírke výške
  • vw - jednotka relatívna k šírke obrazovky

Farby

Zápisy farieb

  • black - anglický názov farby - nepoužíva sa
  • #000000 - hexametrický zápis - najčastejšie sa používa
  • #000 - skrátený hexametrický zápis - možné použiť len pri farbách ktoré majú rovnaké dvojice (#aabbcc -> #abc)
  • rgb(0, 0, 0) - rgba zápis (red, green, blue)
  • rgb(0, 0, 0, .5) - rgba zápis (red, green, blue, alpha) - alpha vyjadruje stupeň priehladnosti 0 - 1 - používa sa pri priehladných farbách
  • rgb(0, 0%, 0%) - hsl zápis (hue, saturation, lightness)
  • rgb(0, 0%, 0%, .3) - hsla(hue, saturation, lightness, alpha) - alpha vyjadruje stupeň priehladnosti 0 - 1

Font

Font family

Je možné použiť voľne dotupné fonty, platené fonty, vytvorený font. Doporučujem používať google fonty, široká paleta fontov na výber, možnosť použitia naprieč rôznymi prehliadačmi. Pokiaľ sa rozhodneme pre výber fontu pri ktorom máme pochybnosti o jeho vykreslení na všetkých prehliadačoch, je vhodné doplniť zástupný font alebo prípadne univerzány serif / sans-serif.

font-family: arial, sans-serif;
  • Vyber si ľubovoľný google font a použi ho na celý web
  • Vyber si ďalší ľubovoľný google font a použi ho na nadpisy

Padding / Border / Margin

Okolo každého objektu je možné stanoviť rozostupy

Padding

Vnútorný rozostup - je súčasťou daného objektu - započítava sa do šírky a výšky. Rozostup sa stanovuje:

  • v jednotkách
padding: 20rem 10rem 15rem 30rem - hodnoty sú stanovené do kolečka v smere hodinových ručičiek o vrchu - vrchný, pravý, spodný, ľavý
padding: 20rem 10rem - vrchný a spodný rozostup bude 20rem, pravý a ľavý bude 10rem
padding: 20rem 10rem 15rem - vrchný rozostup bude 20rem, pravý bude 10rem spodný 15rem a ľavý bude tiež 10rem
padding: 20rem - zo všetkých strán daného objektu bude rozostup 20rem

padding-top: 10rem;
padding-right: 10rem;
padding-bottom: 10rem;
padding-left: 10rem;
  • Všetkým farebným boxom vrchný a spodný padding 2rem a bočný padding 1.75rem

Border

Ohraničenie objektu. Nachadza sa na hranici objektu medzi marginom a paddingom. Je možné stanoviť pre všetky strany alebo len pre niektorú. Štýlujeme hrúbku, prevedenie a farbu a zaoblenie. Ak chceme, aby ma objekt kruhový tvar vytvoríme štvorec nastavíme border-radius na 50%

border: 1px solid #fff
  • Všetkým farebným boxom naštýluj ohraničenie vo farbe daného boxu, ale bez priehladnosti
  • Všetkým farebným boxom naštýluj mierne zaoblen0 rohy

Margin

Vonkajší rozostup - nie je súčasťou daného objektu. Rozostup sa stanovuje:

  • v jednotkách
  • auto pokiaľ poznáme šírku objektu je ho možné pomocou margin auto horizontálne vycentrovať.
margin: 20rem 10rem 15rem 30rem - hodnoty sú stanovené do kolečka v smere hodinových ručičiek o vrchu - vrchný, pravý, spodný, ľavý
margin: 20rem 10rem - vrchný a spodný rozostup bude 20rem, pravý a ľavý bude 10rem
margin: 20rem 10rem 15rem - vrchný rozostup bude 20rem, pravý bude 10rem spodný 15rem a ľavý bude tiež 10rem
margin: 20rem - zo všetkých strán daného objektu bude rozostup 20rem

margin-top: 10rem;
margin-right: 10rem;
margin-bottom: 10rem;
margin-left: 10rem;
  • Všetkým farebným boxom vrchný a spodný margin 20px.
  • Obalovací div vycentruj pomocou marginu.

Display

pomocou štýlu display ovplyvňujeme spôsob zobrazenie prvkov. Niektoré tagy majú preddefinované výchozie zobrazenie.

  • block - objekt má 100% šírku svojho rodiča, pokiaľ vykreslíme za sebou dva takéto objekty, každý sa vykreslí na samostatnom riadku
  • inline - objekt má šírku podľa svojho obsahu, nie je možné mu stanoviť výšku a šírku, pokiaľ vykreslíme za sebou dva takéto objekty, vykreslia sa v jednom riadku
  • inline-block - objekt sa chová ako inline objekt, ale je mu možné stanoviť výšku a šírku
  • flex vysvetlíme si v samostatnej kapitole
  • none objekt sa nevykreslí
display: inline-block;

Visibility

Slúži pre zobrazenie / skrytie objektu

  • visible
  • hidden
vissibility: hidden;

Opacity

Priehladnosť zobrazenia v rozmedzí 0 až 1

  • 0 - plná priehladnosť, objekt nie je vidno
  • 1 - plná nepriehladnosť, objekt je zcela vidno
opacity: .5;

Display none / visibility hidden / Opacity 0

Všetky tri vlastnosti spôsobujú nezobrazenie objektu. Je medzi nimi rozdiel?

  • Pod zadaním sa nachádzajú 3 objekty, obalovaciemu objektu daj zobrazenie flex
  • Objekty naštýluj (farba pozadia, padding, margin)
  • Prvému objektu nastav display: none a skontroluj, čo sa stalo
  • Druhému objektu nastav visibility: hidden a skontroluj, čo sa stalo. Zaznamenal si rozdiel?
  • Tretiemu objektu nastav opacity:0 a skontroluj, čo sa stalo. Zaznamenal si rozdiel?
  • Všetky 3 objekty obsahujú v sebe odkaz, dá sa naň kliknúť?
  • Nastav .answer-box display:block -> je potrebné prepísať inline štýl

  • Display none - objekt nie je vidno, na webe sa nevykreslí (nemá zachované miesto), nie je možné naň kliknúť
  • Visibility hidden - objekt nie je vidno, na webe sa vykreslí (má zachované miesto), nie je možné naň kliknúť
  • Opacity 0 - objekt nie je vidno, na webe sa vykreslí (má zachované miesto), je možné naň kliknúť

Display flex

Jedná sa o relatívne novinku, ktorá významných spôsobom uľahčuje vertikálne a horizontálne zarovnanie objektov v rodičovi. Problematika display flex je pomerne rozsiahla, pre jej pochopenie a plné využívanie doporučejm prejsť CSS-tricks Flexbox.

Štýly rodiča

Display

  • Display: flex

Flex-direction

  • flex-direction: row - potomkovia sa vykreslia v jednom riadku, aj keď majú display: block
  • flex-direction: column- potomkovia sa vykreslia v stĺpci

Justify-content

Upravuje zarovnanie potomkov voči sebe.

  • justify-content: flex-start -potomkovia sa zarovnajú na začiatok rodiča
  • justify-content: flex-end -potomkovia sa zarovnajú na koniec rodiča
  • justify-content: center -potomkovia sa zarovnajú na stred rodiča
  • justify-content: space-between -medzi potomkami sa vytvoria rovnomerné medzery. Prvý potomok je zarovnaný na začiaku rodiča, posledný na koniec rodiča
  • justify-content: space-around -medzi potomkami sa vytvoria rovnomerné medzery

Align-items

Upravuje zarovnanie potomkov voči rodičovi

  • align-items: flex-start -potomkovia sa zarovnajú na začiatok rodiča
  • align-items: flex-end -potomkovia sa zarovnajú na koniec rodiča
  • align-items: center -potomkovia sa zarovnajú na stred rodiča

Štýly potomkov

Je možné zarovnať aj samostatne jednotlivých potomkov - samoštúdium

  • Pod zadaním sa nachádzajú ukážkové objekty, obalovaciemu objektu daj zobrazenie flex, nastav mu farbu pozadia a min. spodný margin
  • Objekty naštýluj (farba pozadia, padding, margin)
  • Obom skupinám nastav zobrazenie flex
  • Jednej skupine nastav zarovnanie do riadku a druhej do stĺpca

Position

Pozícia objektu

  • static - defaltná hodnota, objekt sa vykreslí v takom poradí aké ma v kóde
  • relativ - objekt má pozíciu relatívnu voči svojej bežnej pozícii
  • absolut - obejkt má relatívnu pozíciu voči prvému rodičovy, ktorý nemá pozíciu static
  • fixed - obejkt má relatívnu pozíciu voči prehliadaču

Presnú pozíciu stanovujeme pomocou:

  • top
  • right
  • bottom
  • left

Tip

Pokiaľ chceme objekt s position:absolut vycentrovať na stred je dobré použiť na relevantného rodiča display:flex

  • Pod zadaním sa nachádzajú ukážkové objekty, obalovaciemu objektu nastav farbu pozadia a min. spodný margin
  • Objekty naštýluj (farba pozadia, padding, margin)
  • Objektom priraď danú pozíciu a nastav ľubovoľne hodnoty top, right, bottom, left a všimni si ako sa vykreslia
  • Obalovaciemu objektu nastav position:relativ. Nie+co sa zmenilo
  • Na oboch skupinách si vyskúšaj vlastnosť justify-content, align-items

Obrázky

Rozmer obrázkov

Pokiaľ chceme vykresliť obrázok so zachovaním pomeru strán, nastavíme požadovanú výsku alebo šírku a druhý rozmer stanovíme na auto

Pokiaľ chceme vykresliť vykresliť obrázok v presnom rozmere bez toho aby sa zdeformoval použijeme:

  • object-fit: cover; - defaltná hodnota, objekt sa vykreslí v takom poradí aké ma v kóde
  • object-position: 50% 50%; - defaltná hodnota, objekt sa vykreslí v takom poradí aké ma v kóde
  • Na testovacích obrázkoch si vyskúšaj rozdielne vykreslenie

Obrázky na pozadí

Okrem farby je možné stanoviť ako pozadie aj obrázok. Pri obrázkoch na pozadí nastavujeme:

  • background-image: - cesta k obrázku
  • background-position: - pozícia obrázka cover/contain
  • background-repeat: - opakovanie obrázka
  • .img-box nastav testovací obrázok ako pozadie a naštýluj ho

Hover

Pri niektoré objekty chceme, aby po tom, co na ne užívateľ nascrolluje myš zmenili štýl a tým dali užívateľovi najavo, že je možné na ne kliknúť - button, link

  • .link:hover
  • Naštýluj odkazov hover stav
